## 👾 About

A fully responsive  personal portfolio built with pure HTML, CSS, and JavaScript. Designed to stand out with neon glows, animated elements, and a futuristic dark aesthetic.

---

## ✨ Features

- 🎨 **Cyberpunk Design** — Deep purple & neon color palette with scanline effects
- ⚡ **Custom Cursor** — Glowing neon cursor with ring animation
- 💬 **Typing Animation** — Dynamic text in the hero section
- 🔄 **Scroll Reveal** — Sections animate in as you scroll
- 📱 **Fully Responsive** — Works on all screen sizes
- 🌐 **Grid Background** — Animated cyberpunk grid overlay
- 🔗 **Live Project Links** — Direct links to deployed projects
